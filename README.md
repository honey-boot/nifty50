# 📊 Data-Driven Nifty 50 Stock Analysis Dashboard

A complete end-to-end data analytics project to analyze and visualize the performance of Nifty 50 stocks. This project includes data cleaning, transformation, statistical analysis, and dashboarding using **Streamlit**, **SQL (TiDB Cloud)**, and **Power BI**.

---

## 🚀 Project Overview

This project delivers a powerful stock analysis dashboard to help investors, traders, and analysts gain insights into Nifty 50 companies' trends over the past year. It includes:

- Data extraction from YAML files and conversion to structured CSV
- SQL integration with TiDB Cloud for scalable storage and querying
- Analytical insights (volatility, returns, sector trends, correlation)
- Interactive dashboard using Streamlit
- Professional data visualization with Power BI

---

## 🧩 Features and Use Cases

| Use Case | Description |
|----------|-------------|
| 📈 **Top Performers** | View top 10 green (gaining) and red (losing) stocks based on yearly returns |
| 💹 **Volatility Analysis** | Identify the most volatile stocks using standard deviation of returns |
| 📊 **Cumulative Returns** | Track performance growth of top 5 stocks throughout the year |
| 🏭 **Sector-wise Performance** | Average yearly return by sector |
| 🔗 **Stock Correlation** | Heatmap of stock price correlations |
| 📆 **Monthly Trends** | Top 5 gainers and losers each month |

---

## 🛠 Tech Stack

- **Python**: Data cleaning and transformation
- **Pandas, NumPy**: Analysis
- **SQLAlchemy + MySQL / TiDB**: Database storage and querying
- **Streamlit**: Real-time dashboard interface
- **Power BI**: Additional visual analytics
- **Matplotlib / Seaborn**: Optional plotting
